# **App Name**: PrimePanel

## Core Features:

- Admin Authentication: Secure admin login/logout functionality powered by Firebase Authentication.
- Sidebar Navigation: Intuitive sidebar navigation with icons for main sections (Dashboard, Posts, Scores, Users, Settings).
- Dashboard Analytics: Display analytics cards summarizing key metrics (Total Users, Active Sessions, Posts Published, Live Matches).
- Posts Management: Manage blog posts within a table (title, author, status, date) with edit, delete, and publish/unpublish options.
- Scores Display: View live and upcoming sports matches (Cricket and Football) with tabbed or filterable categories.
- User Management: Searchable and filterable list of users from Firebase Authentication, showing profile picture, name, email, and role.
- Settings Control: Admin settings page for toggling the UI theme (dark/light).

## Style Guidelines:

- Primary color: #5046E5 (RGB) - A modern, techy purple, suggesting sophistication, control, and insights, avoiding the clich√© of overused blues and greens.
- Background color: #0A0929 (RGB) - Dark background enhances contrast and gives a professional feel.
- Accent color: #FF66C4 (RGB) - A vibrant pink, contrasting the primary hue for key actions, creating focus and memorable interactions.
- Font: 'Inter' (sans-serif) for both headlines and body text to maintain a consistent, clean, and modern look.
- Use a set of consistent, simple line icons throughout the interface to match the clean design.
- Emphasize rounded cards, soft shadows, and clear section separation to create a user-friendly and visually appealing dashboard layout using Tailwind CSS and Shadcn UI.
- Incorporate subtle transitions and animations for UI elements and data updates to improve user experience without being distracting.
